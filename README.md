# Multimodal-Data-Exploration-Educational-Data-CIS-635-

This repo contains the data and notebook used for our final project for CIS 635. Our project will explore the relationship between school funding and dropout rates using publicly available data. We’re sourcing our data specifically through Census and MI school data. By fusing multiple datasets and incorporating Python-based data analysis and visualization techniques, patterns linking finances and student retention will be uncovered. The results would highlight how data fusion can reveal hidden correlations between educational performances. Finally, the project will reflect on how artificial intelligence could automate heterogeneous information management to flag at-risk information and support fair decision-making.
